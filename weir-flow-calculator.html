① 矩形薄壁堰（Rehbock 公式）

自由出流基本式：

Q = m₀ · ε · σ · b · √(2g) · H^(3/2)

其中 m₀ 为流量系数,ε 为侧收缩系数,σ 为淹没系数（自由出流时=1）。

Rehbock 流量系数（无侧收缩,B=b）：

m₀ = 0.403 + 0.053 · H/P + 0.0007/H

适用条件：0.03 ≤ H ≤ 0.75 m;H/P ≤ 5;P ≥ 0.1 m。

有侧收缩（B > b）：采用 Francis 修正 或 ε 系数,工具中取：

ε = 1 − 0.1·n·H/b（n 为每边收缩数）

② 三角堰（V形堰,Kindsvater-Shen）

Q = (8/15) · C_d · tan(θ/2) · √(2g) · H^(5/2)

90° 三角堰是量水最常用的（小流量精度高）。C_d ≈ 0.585（标准 Thomson 公式）。

Q = 1.4 · H^(5/2) (90° 简化式, Q m³/s, H m)

适用：0.05 ≤ H ≤ 0.38 m; P ≥ 2H; 上游渠宽 ≥ 3H。

③ 梯形堰 Cipoletti（边坡 1:4）

特点：侧边向外倾斜，补偿侧收缩损失，流量系数近似为常数。

Q = 1.86 · b · H^(3/2) (Cipoletti 公式)

即 m₀ ≈ 0.42,ε ≈ 1.0（自动补偿）。适用：0.06 ≤ H ≤ 0.60 m;b ≥ 3H;P ≥ 2H。

④ 宽顶堰

Q = m · b · √(2g) · H₀^(3/2)

其中 H₀ = H + v₀²/(2g) 为包含行近流速头的全水头,m 为流量系数:

δ/H堰顶边缘形式m 值
2.5-4直角边0.32-0.35
2.5-4圆角修圆0.36-0.385
2.5-10一般宽顶堰0.32-0.36

本工具按 δ/H 判别,默认取 0.36（圆角修圆）。

⑤ 淹没出流修正

当下游水位 h_s > 0（下游水面高于堰顶）时:

h_s / H_0 > 0.15（宽顶堰） 或 h_s/H > 临界值（薄壁堰）,进入淹没状态

淹没系数 σ 查表或用经验公式计算,本工具采用:

σ = (1 − (h_s/H)ⁿ)^0.5 (简化经验式)

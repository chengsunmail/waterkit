① Manning 公式（谢才-曼宁,均匀流）

Q = A · v = (1/n) · A · R^(2/3) · i^(1/2) v = (1/n) · R^(2/3) · i^(1/2) = C · √(R · i)

C = R^(1/6)/n 为谢才系数,n 为曼宁粗糙系数。

② 各断面水力要素

梯形断面（底宽 b,边坡 m = 水平:竖直,水深 h）：

A = (b + m·h) · h χ = b + 2h·√(1+m²) R = A / χ B = b + 2m·h（水面宽）

矩形断面（相当于 m=0 的梯形）：

A = b · h, χ = b + 2h, R = bh / (b+2h), B = b

圆形断面（管径 D,水深 h）：

θ = 2 · arccos(1 − 2h/D) [弧度] A = D²/8 · (θ − sin θ) χ = D · θ / 2 R = A / χ B = D · sin(θ/2)（水面宽）

③ 临界水深与临界坡度

α · Q² · B / (g · A³) = 1 → 求 h_c

α 动能修正系数取 1.0,g = 9.81 m/s²。本工具采用二分法迭代求解。

i_c = n² · Q² / (A² · R^(4/3)) 代入 h = h_c

④ 弗劳德数与流态

Fr = v / √(g · A / B) = v / √(g · h_m)

h_m = A/B 为平均水深。Fr < 1 缓流(h > h_c),Fr = 1 临界流,Fr > 1 急流(h < h_c)。

⑤ 水力最佳断面

矩形：h = b/2（宽深比 β = b/h = 2）

梯形：β = b/h = 2·(√(1+m²) − m)

例如 m=1.5 时,最佳 β ≈ 0.606,即 b/h ≈ 0.606。

圆形：充满度 h/D = 0.813 时流量最大（非满流效应）。

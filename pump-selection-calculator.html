在泵厂样本中按此流量、扬程选择，优先工作点落在高效区（Q_opt 的 80-110%）。

① 系统扬程组成

H = Hg + Hp + hw + H_margin

Hg = z₂ − z₁（出水液面高 − 吸水液面高，m）
Hp = (P₂ − P₁)/(ρ·g) × 100（两侧表压差换算成液柱，m）
hw = Σhf + Σhj（沿程损失+局部损失）
H_margin：富余量，一般取总扬程的 5-15%。

② 功率计算

Pe = ρ·g·Q·H / 1000 / 3600 [kW]（水功率） Psh = Pe / η_pump / η_trans [kW]（轴功率） Pm = K · Psh / η_motor [kW]（配套电机）

简化公式（清水 ρ=1000, g=9.81）：

P_e = Q·H / 367 [kW] （Q in m³/h, H in m）

③ NPSH 汽蚀校核

NPSHa = (Pa − Pv)/(ρ·g) × 100 − Hs − hf_s [m]

Pa 大气压（按海拔），Pv 饱和蒸汽压（按水温），Hs 吸水几何高度（吸上为正），hf_s 吸水管水头损失。

NPSHa ≥ NPSHr + 安全余量（0.5~1.0 m）

最大允许吸水高度：

Hs_max = (Pa − Pv)/(ρ·g)·100 − NPSHr − hf_s − 安全余量

④ 泵相似律（变频调速预估）

对同一台泵在不同转速 n 下：

Q' / Q = n' / n H' / H = (n' / n)² P' / P = (n' / n)³

⚠️ 降频后扬程快速降低，如系统有较大静扬程，过度降频会导致流量为零。
